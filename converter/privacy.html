Currency Converter — Privacy Policy

Last updated: March 25, 2026

Overview

Currency Converter is a currency conversion app built by Yousif Ameen under Approve The Move.

Data Storage

Currency Converter stores only cached exchange rates on your device for offline use. No personal data is stored.

Data Collection

Currency Converter collects no personal data. Specifically:

  • No analytics or usage tracking
  • No advertising frameworks
  • No account creation
  • No personal information collected or transmitted

Network Activity

The app makes a single type of network request: fetching current exchange rates from a public API (open.er-api.com). This request contains no personal data, device identifiers, or tracking information. It only retrieves currency exchange rates.

Changes to This Policy

If this privacy policy ever changes, you will be notified within the app before any changes take effect.

Contact

If you have questions about this privacy policy, contact: support@approvethemove.com
