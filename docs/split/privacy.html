ATM Split — Privacy Policy

Last updated: March 25, 2026

Overview

ATM Split is an expense splitting app built by Yousif Ameen under Approve The Move. Your privacy is fundamental to how this app is built.

Data Storage

All your data — groups, members, expenses, and settlements — is stored locally on your device. No cloud sync, no external servers.

Data Collection

ATM Split collects no data. Specifically:

  • No analytics or usage tracking
  • No advertising frameworks or identifiers
  • No account creation or sign-in required
  • No third-party SDKs that collect data
  • No server-side storage

Network Activity

The only network request ATM Split makes is fetching exchange rates from a public API for multi-currency support. This request contains no personal data — it only retrieves currency rates.

CSV Export

When you export group data as CSV, the file is generated locally on your device. You control where it goes via the iOS share sheet.

Data Deletion

You can delete any group, expense, or member at any time. Uninstalling the app removes all data permanently.

Changes to This Policy

If this privacy policy ever changes, you will be notified within the app before any changes take effect.

Contact

If you have questions about this privacy policy, contact: support@approvethemove.com
