Qatari Joker — Privacy Policy

Last updated: April 2, 2026

Overview

Qatari Joker is a card game app built by Yousif Ameen under Approve The Move.

Data Storage

Qatari Joker stores only your app preferences (such as theme, accent color, player name, and game settings) locally on your device using UserDefaults. This data never leaves your device. All game state exists only in memory while you play. When you close the app, the game state is gone. There is no cloud sync.

Data Collection

Qatari Joker collects no personal data. Specifically:

  • No analytics or usage tracking
  • No advertising frameworks
  • No account creation
  • No server-side communication

Multiplayer

Local multiplayer uses Apple's MultipeerConnectivity framework to connect nearby devices directly over Wi-Fi and Bluetooth. No internet connection is used. The only data transmitted is game state (card plays, bids, scores) between connected players during a session.

Changes to This Policy

If this privacy policy ever changes, you will be notified within the app before any changes take effect.

Contact

If you have questions about this privacy policy, contact: support@approvethemove.com
